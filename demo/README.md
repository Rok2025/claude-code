# Demo Project

这是一个演示项目，用于测试 Claude Code 的 `/init` 命令。

## 功能特性

- Node.js + Express 后端
- TypeScript 支持
- Jest 测试框架
- ESLint 代码检查

## 快速开始

```bash
npm install
npm run dev
```

## 项目结构

```
demo/
├── src/
│   ├── index.ts
│   ├── routes/
│   └── utils/
├── tests/
├── package.json
└── README.md
```
